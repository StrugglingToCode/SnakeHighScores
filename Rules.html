Official Rules

Section 1: Submission of A Highscore in the Standard Category

Part A: Submitting Video Highscores

1)Video must be uncut
2)Video must include the run in its entirety
3)Video must include final score

Part B: Submitting Picture Highscores

1)Picture must include the entirety of the browser window
2)Picture must show the modifiers used

Part C: Submitting Highscores With Witnesses

1)Run requires three(3) witness testimonies
2)Witnesses must watch the entirety of the submitted run

Part D: Requirements for a valid Highscore

1)All settings except for Snake Colour and Fruit Skin must be default
2)Volume and Sound settings can be changed
3)The run must be played on the Google Chrome Web Browser

Section 2: Submission of A Highscore in the Sped Up Category

Part A: Submission of Highscore

1)All settings except Snake Colour, Fruit Skin, and Snake Speed must be default
2)Run must comply by all rules under Section 1 with the exception of Section 1: Part D: 1)All settings except for Snake Colour and Fruit Skin must be default

Section 3: Punishments

Part A: UnsportsmanLike Actions

1)Interruption of Playthrough causing the run to fail
i)If the run is ended prematurely and the score is under fifty (50) the offending party will receive a formal warning from the moderation team
ii)If the run is ended prematurely and the score is over fifty (50) but more than ten (10) less than the current record the offending party will be banned from submitting a run for twenty four (24) hours following the report
iii)If the run is ended prematurely and the score is within ten (10) points of the current record the offending party will be banned from submitting a run for seventy two (72) hours following the report

Section 4: Participation Requirements

Part A: Specifications

1)The runner must attend the Energetic Learning Campus at the time of submissions
